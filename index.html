About Me

Hello, I'm a Charlotte-based web designer and developer, who enjoys writing code and creating innovative web pages and apps.

I'm attending UNCC Coding Bootcamp and learning new programming languages and best practices, that will help me master my coding skills and become a full-stack web developer.

Right now, my focus is writing clean, well-formatted code by hand to make sure that the content is easy to read, easy to collaborate, troubleshoot and accessible.

Portfolio


Crystals Collector is a browser game. The player has to collect the right amount of the different crystals to match the given number and win the game.

Technologies Used: HTML, CSS, jQuery

Train Scheduler app incorporates Firebase to host arrival and departure data, and manipulates it with Moment.js. The app provides up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at the station.

Technologies Used: HTML, CSS, JavaScript, jQuery, Firebase, Moment.js, Bootstrap

LIRI is a command line node app that takes in parameters and gives back data. LIRI searches Bands in Town for concerts, Spotify for songs, and OMDB for movies.

Technologies Used: Node.js, JavaScript, npm packages: Node-Spotify-API, Axios, Moment & DotEnv

Psychic Game randomly picks a letter, and the user has to guess the letter.

Technologies Used: JavaScript

Bardify app processes the lyrics of your choice through Shakespearan translator and brings them back to you via text to speech. This was a group project.
My role included app branding and dynamically generating buttons, forms and lyrics with jQuery, that makes smooth transitions between different screens.

Technologies Used: JavaScript, jQuery, Firebase, APIs, Bulma

Giphy is a dynamic web application that populates with the GIFs of user's choice. The app makes AJAX calls to the Giphy API and utilizes JavaScript and jQuery to manipulate the HTML on the DOM.

Technologies Used: HTML, CSS, JavaScript, jQuery, AJAX, APIs

Trivia Game is a quiz with multiple choice options. The player has a limited amount of time to finish the quiz. The final page reveals the number of players correct and incorrect answers.

Technologies Used: HTML, CSS, JavaScript, jQuery
